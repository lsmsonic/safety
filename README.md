# 🛡️ AI 안전보건 위험성 평가 도구 (AI Safety Risk Assessment Tool)

현장 사진 한 장으로 위험요소 식별, 위험 점수 산정(빈도 × 강도), 관련 법령 매핑 및 단계별 개선 조치 방안을 자동 생성하는 AI 기반 웹 애플리케이션입니다.

본 도구는 산업안전보건법 제36조(위험성평가) 및 중대재해처벌법 제4조의 안전보건 확보 의무 이행을 돕는 보조 참고 도구로 설계되었습니다.

---

## 📂 주요 기능

1. **AI 기반 현장 분석**:
   - 업로드한 이미지(카메라 촬영, 갤러리 선택, 드래그 앤 드롭 지원) 분석.
   - Google Gemini API 또는 Anthropic Claude API와 연동하여 분석을 수행합니다.
2. **정량적 위험 점수 평가**:
   - 빈도(1~5)와 강도(1~5)의 곱으로 위험도 점수(1~25점)를 자동 산정하여 4단계 등급(`낮음`, `보통`, `높음`, `매우 높음`)으로 진단합니다.
3. **양방향 이미지 핀 매핑**:
   - 분석 결과 카드와 사진 위 번호 핀이 서로 링크되어 있어, 핀 클릭 시 카드 하이라이팅 및 카드 내 `[📍 사진에서 보기]` 지원.
4. **개선조치 이행 체크리스트**:
   - AI가 제안한 즉시/단기/중기 조치 사항의 조치 여부를 체크박스로 기록하고 실시간 조치율을 관리할 수 있습니다.
5. **통계 분석 및 기록 데이터 저장**:
   - 브라우저 로컬 저장소(`localStorage`)를 이용해 모든 진단 데이터를 안전하게 보관하고 등급별/유형별 통계를 시각화합니다.
6. **보고서 출력 (인쇄)**:
   - 결재란과 공지문이 포함된 A4 규격의 공식적인 보고서 레이아웃을 통해 바로 인쇄하거나 PDF로 저장할 수 있습니다.
7. **데모 모드**:
   - API 키가 없어도 건설현장, 기계공장, 화학물질 시나리오의 가상 데이터를 기반으로 기능 시연이 가능합니다.

## 🛠️ 설정 및 구동 방법

이 애플리케이션은 **단일 파일로 작동하는 Serverless HTML5 SPA**입니다. 별도의 서버나 빌드 환경이 불필요합니다.

### 1단계: API 키 발급
실제 AI 분석을 사용하려면 다음 서비스 중 하나에서 API 키를 무료로 발급받아야 합니다.
* **Google Gemini API**: [Google AI Studio](https://aistudio.google.com/)에서 발급 (무료 요금제 지원)
* **Anthropic API**: [Anthropic Console](https://console.anthropic.com/)에서 발급

### 2단계: 웹 구동 및 설정
1. `index.html` 파일을 더블 클릭하여 브라우저에서 직접 열거나 웹 서버에 업로드하여 접속합니다.
2. 하단 내비게이션의 **[설정]** 탭으로 이동합니다.
3. **API 제공사**를 선택하고 발급받은 **API 키**를 입력한 후 저장합니다.
4. 분석 탭으로 돌아가 사진을 업로드하고 **[AI 위험성 분석 시작]** 버튼을 누릅니다.
